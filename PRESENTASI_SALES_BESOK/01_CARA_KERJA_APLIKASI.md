### 5. WORKFLOW HARIAN

```

3. Cek dashboard:
   - Stok menipis? Restok

**Siang (Transaksi):**
```
1. Customer datang

```

**Malam (Tutup Toko):**
```

**Untuk Efisiensi:**
- Input semua produk dengan barcode

- Set alert stok minimum
- Backup data rutin
- Update harga berkala

### 7. TROUBLESHOOTING UMUM

**Masalah: Barcode tidak terbaca**
- Solusi: Pastikan barcode jelas, tidak rusak
- Coba scan ulang dengan angle berbeda
- Atau input manual

**Masalah: Printer tidak cetak**
- Solusi: Cek koneksi printer
- Cek kertas struk
- Restart printer
- Test print dari pengaturan

**Masalah: Aplikasi lemot**
- Solusi: Tutup aplikasi lain
- Restart aplikasi
- Cek koneksi internet
- Clear cache

**Masalah: Lupa password**
- Solusi: Klik "Lupa Password"
- Cek email untuk reset link
- Buat password baru
