## Langkah 3: Pilih Produk & Isi Form

1. **Klik tombol "Restock"** pada produk yang ingin dipesan
2. Modal akan muncul dengan form:

### Bagian 1: Informasi Supplier
- **Nama Supplier** (wajib)
  - Contoh: "Toko Grosir ABC"
  
- **No. WhatsApp Supplier** (wajib)
  - Format: 628123456789 (tanpa +, tanpa spasi)
  - Contoh: 628123456789
  
- **Alamat Supplier** (opsional)
  - Contoh: "Jl. Pasar Besar No. 123, Jakarta"

### Bagian 2: Detail Pesanan
- **Jumlah Pesanan** (wajib)
  - Masukkan jumlah yang ingin dipesan
  - Contoh: 50
  
- **Catatan untuk produk** (opsional)
  - Contoh: "Warna merah, ukuran L"
  
- **Catatan tambahan** (opsional)
  - Contoh: "Mohon dikirim sebelum akhir bulan"

## Langkah 4: Buat Surat Pesanan

1. Klik tombol **"Buat Surat Pesanan"** (warna merah)
2. Akan muncul dialog dengan 4 pilihan:

### Pilihan 1: Export PDF
- Membuka surat dalam format PDF
- Bisa langsung dicetak (Ctrl+P)
- Atau save as PDF
- **Catatan**: Hanya tersedia di versi web

### Pilihan 2: Salin Teks
- Copy surat pesanan ke clipboard
- Bisa paste ke WhatsApp, Email, atau platform lainnya
- Tinggal paste dan kirim
- **Tersedia di semua platform**

### Pilihan 3: Simpan Gambar
- Simpan surat pesanan sebagai gambar
- Fitur akan segera tersedia

### Pilihan 4: Batal
- Kembali ke form jika ada yang salah

## Langkah 5: Kirim ke Supplier

### Jika Pilih Salin Teks:
1. Surat pesanan otomatis tersalin ke clipboard
2. Buka WhatsApp supplier
3. Paste (Ctrl+V atau tahan lalu pilih Paste)
4. **Cek sekali lagi** isi pesanan
5. Klik tombol **Send** (âœ“)

2. Cek folder Downloads
3. File bernama: Surat-Pesanan-RST-xxxxx.pdf
4. Bisa langsung dibagikan via email/WhatsApp atau dicetak

### Jika Pilih Simpan Gambar:
1. File PNG akan otomatis terdownload
2. Cek folder Downloads

## FAQ

**Q: Kenapa produk saya tidak muncul di daftar?**
A: Fitur ini hanya menampilkan produk dengan stok â‰¤10 unit. Jika produk Anda stoknya masih banyak, tidak akan muncul.

**Q: Bisa pesan beberapa produk sekaligus?**
A: Saat ini satu surat untuk satu produk. Untuk beberapa produk, buat surat terpisah atau gabungkan manual.

**Q: Format nomor WhatsApp yang benar?**
A: 628123456789 (kode negara 62, tanpa +, tanpa 0 di depan, tanpa spasi)

**Q: PDF tidak muncul?**
A: Pastikan browser tidak memblokir popup. Biasanya ada notifikasi di address bar untuk allow popup.

**Q: Bisa edit surat setelah dibuat?**
A: Tidak bisa. Jika ada kesalahan, buat surat baru dengan data yang benar.

**Q: Surat tersimpan di mana?**
A: Surat tidak tersimpan otomatis. Jika perlu arsip, save PDF atau screenshot chat WhatsApp.
